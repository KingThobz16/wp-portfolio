Sport For Lives

The Sport For Lives website is a fundraising platform specializing in sport-based donations and pledging, where physical activities are organized to raise funds for various causes. Built using WordPress, WooCommerce, Elementor, and the Krowd theme, the site allows donors to either contribute directly to campaigns or pledge specific amounts based on performance metrics. For instance, donors can pledge R10 per goal scored or R100 per kilometer run.

I was responsible for extending the site’s core functionality to support these dynamic pledges, enabling users to make performance-based commitments for any campaign type. I developed a seamless pledge capture modal that aligned with the site’s aesthetics and provided fundraisers with real-time access to pledge data on the frontend. This data could be used to send custom or bulk emails to pledges at the end of the campaign.

I leveraged technologies such as PHP, MySQL, HTML, CSS, JavaScript, PHPMailer, AJAX, and jQuery to implement these customizations, modifying multiple plugins and the main theme to meet the client’s requirements. Additionally, I ensured the site's security and stability through regular maintenance, malware hardening, and applying best security practices. I also provided ongoing support to the client, addressing any site-related queries and ensuring smooth operations.
